# Crates
Crates is a rewrite of the expeditions plugin.

## Commands:

|                                         Command | Permission        | Description                                           |
|------------------------------------------------:|-------------------|-------------------------------------------------------|
| /crates                                    | None              | Opens the crates menu                                 |
| /cratesadmin                               | crates.admin | Shows all admin commands                              |
| /cratesadmin givepremium <player> [amount] | crates.admin | Give a premium crate to a player, default amount is 1 |
| /cratesadmin reload                        | crates.admin | Reload the configuration/loot tables                  |
